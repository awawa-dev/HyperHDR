Metadata bucket LUT files

These fields drive metadata-driven LUT selection. HDR and Dolby Vision both split into MDL1000 and MDL4000 families, but they do not use the same metadata path. HDR buckets use HDR static metadata, while Dolby Vision buckets use Dolby Vision source metadata for family selection and Dolby Vision L1 or FLL runtime values for bucket selection. Blank HDR buckets fall back to the shared HDR LUT. Blank DV MDL4000 entries fall back to the matching MDL1000 entry before the shared LLDV LUT.

These extra actions do not replace the bottom save-and-apply button. Use them to persist or test transfer and metadata-bucket changes without forcing the explicit LUT reload path.
