CEC / eARC / ARC CONTROL:
VRRoom has three separate CEC zones to support CEC commands and ARC/eARC communication.
MAIN ZONE CEC controls channel switching and eARC/ARC retrieval from the TX0 connected display
AUDIO OUT zone controls the volume and power up/down of the AUDIO OUT connected AVR/Soundbar
eARC TX zone controls the volume and power up/down of the eARC TX connected AVR/Soundbar as well as eARC/ARC transmission

All changes to CEC/eARC/ARC parameters require a VRROOM power cycle. Unplug/replug power

MAIN ZONE:
MAIN ZONE CEC must be enabled to support CEC channel switching and eARC/ARC retrieval
eARC/ARC support requires "Audio System", make sure no other AVR devices are connected to the TV
eARC can support lossless bitstream formats, such as MAT atmos, TrueHD, DTS:X, PCM 7.1
ARC can support up to of DD+ Atmos, stereo PCM

CEC is enabled when the checkbox is selected

eARC/ARC:
AUTO relies on TV CEC commands for switching between eARC/ARC and HDMI. Select the auto mode according to TV capabilities
If the display supports eARC, select Manual eARC. All audio will be routed via TV eARC
If the display supports ARC only, select Manual ARC. All audio will be routed via TV ARC
ARC is max DD+. Consider EDID settings to not exceed the max capability from the HDMI input

SPEAKER MAP
Some TVs will incorrectly stream mark 7.1 for all eARC content with more speakers than stereo
It usually results in 5.1 speaker systems to lose the surrounds
Use the Force 5.1 -setting to correct the speaker mapping in such systems

MUTE MODE
MUTE from TV/source remotes can use dedicated CEC mute on/off commands or a toggle command
Some TVs will cause the toggle command to lose track if pressed many times too quickly

eARC TX ZONE:

OUT MODE:
If the connected device only supports ARC, choose the Always ARC-option, otherwise leave as default
When using Always ARC-option make sure input port AUDIO CAPS don't exceed soundbar/AVR capabilities
Typical ARC soundbar supports maximum DD+ and stereo PCM. Use ARC/BEAM1 or ARC/DD+ AUDIO CAPS

AUDIO OUT ZONE:

POWER ON:
POWER OFF:
VRRoom issues automated power on/off CEC commands to the AUDIO OUT connected sink.
Request Audio Mode = Instructs the sink to enter System Audio mode and to enable volume control
Request Audio Mode + Routing Change = As above but also issues Routing Change to switch to the VRRoom input

INPUT PORT CEC CONTROL:
VRRoom has a capability to individually disable CEC from inputs.
When disabled, no CEC commands will be allowed from the respective input.

Checked means enabled. Default is checked.
